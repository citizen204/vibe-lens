## 🚀 快速开始

### 在线使用

直接访问 → **[vibe-lens.vercel.app](https://vibe-lens.vercel.app)** *(部署后填入)*

### 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/your-org/vibe-lens.git
cd vibe-lens

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
# → http://localhost:3000
```

### 生成项目目录树

```bash
# macOS / Linux
tree -I 'node_modules|.git|.next|dist' --dirsfirst -a

# 或使用 find
find . -not -path '*/node_modules/*' -not -path '*/.git/*' | sort

# Windows (PowerShell)
tree /F /A | clip
```

> **技巧**：将 `tree` 命令输出直接粘贴到 Vibe-Lens 输入框，点击「开始分析」即可。
