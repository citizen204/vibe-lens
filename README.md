## рҹҡҖ Quick Start

### Use Online

Visit вҶ’ **[vibe-lens-zeta.vercel.app](https://vibe-lens-zeta.vercel.app)**

Two ways to analyze a project:

| Mode | How to use |
|------|-----------|
| рҹҗҷ **GitHub URL** | Switch to the "GitHub URL" tab, paste the repo URL, click "Fetch & Analyze" |
| рҹҢІ **Directory Tree** | Switch to the "Directory Tree" tab, paste `tree` command output, click "Analyze" |

### Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/citizen204/vibe-lens.git
cd vibe-lens

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
# вҶ’ http://localhost:3000
```

### Generate a Directory Tree

```bash
# macOS / Linux
tree -I 'node_modules|.git|.next|dist' --dirsfirst -a

# Or with find
find . -not -path '*/node_modules/*' -not -path '*/.git/*' | sort

# Windows (PowerShell)
tree /F /A | clip
```

> **Tip**: Paste the `tree` output directly into the Vibe-Lens input box and click "Analyze".
