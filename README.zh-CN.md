## 🚀 快速开始

### 在线使用

直接访问 → **[vibe-lens-zeta.vercel.app](https://vibe-lens-zeta.vercel.app)**

支持两种分析方式：

| 方式 | 操作 |
|------|------|
| 🐙 **GitHub URL** | 切换到「GitHub URL」标签，粘贴仓库地址，点击「拉取并分析」 |
| 🌲 **目录树** | 切换到「目录树」标签，粘贴 `tree` 命令输出，点击「开始分析」 |

### 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/citizen204/vibe-lens.git
cd vibe-lens

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
# → http://localhost:3000
```

### 生成项目目录树

```bash
# macOS / Linux
tree -I 'node_modules|.git|.next|dist' --dirsfirst -a

# 或使用 find
find . -not -path '*/node_modules/*' -not -path '*/.git/*' | sort

# Windows (PowerShell)
tree /F /A | clip
```

> **技巧**：将 `tree` 命令输出直接粘贴到 Vibe-Lens 输入框，点击「开始分析」即可。
